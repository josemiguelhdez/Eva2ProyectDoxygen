Descripción detallada

The ParameterControlManager handles an array of ParamAdaption instances which dynamically adapt single properties of an object. To make single parameter adaption available for an object (e.g. Problem or Optimizer instance) add a member of type ParameterControlManager. The corresponding getter and setter may either return the ParameterControlManager itself or the array of singleAdapters directly. In the latter case, the GUI is simpler (because theres one fewer window layer) but there must be an additional method getParamControl implemented which returns the ParameterControlManager to make it available to the Processor.
