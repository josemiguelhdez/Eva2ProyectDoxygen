Descripción detallada

Selection methods have to implement on the one hand the selection of the mother and on the other hand the selection for the fathers (eventually more than one). This allows niching method based on mating restrion, which is currently not implemented. All selection method should obey Deb's constraint handling principle, first select feasible, only if all are infeasible select the individuals with the smallest constraint violation.
