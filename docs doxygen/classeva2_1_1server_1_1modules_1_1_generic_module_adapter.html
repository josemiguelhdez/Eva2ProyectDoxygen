EvATabbedFrameMaker eva2.server.modules.GenericModuleAdapter.getModuleFrame ( )

This method returns a newly created GUI element containing the EvA button panel and two JParaPanel instances representing the module parameters (e.g. GO) and the statistics parameters. If the statistics are not of type StatisticsWithGUI, it is assumed that no GUI is desired (and this method should not be called).
