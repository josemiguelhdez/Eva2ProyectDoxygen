double  data[][],
boolean  obeyConst 
)
virtual

This method computes the selection probability for each individual in the population. Note: Summed over the complete population the selection probability sums up to one. Keep in mind that fitness is always to be minimizied! Small values for data => big values for selectionprob.

Parámetros
populationThe population to compute.
dataThe input data as double[][].
