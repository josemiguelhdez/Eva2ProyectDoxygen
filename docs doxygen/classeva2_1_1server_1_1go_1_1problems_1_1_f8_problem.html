void eva2.server.go.problems.F8Problem.initListOfOptima ( )

We identify a set of 2*n+1 optima, which are the global and the first level of local optima. Interestingly, the local optima in the first sphere (close to (1,0,0,0,...) vanish starting with n=18 (checked with Matlab). The next sphere contains O(n^2) optima (I think 3*n*(n-1)). This is unfortunately not mentioned in the papers by Shir and Bäck who still seemed to be able to find 2*n+1...
