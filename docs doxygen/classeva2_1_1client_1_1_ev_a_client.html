Main constructor of the EvA2 client GUI. Works as standalone verson locally or as client for the EvA2 server. GO parameters may be loaded from a file (paramsFile) or given directly as a java instance. Both may be null to start with standard parameters. If both are non null, the java instance has the higher priority. Note that the EvAClient initialized multi-threaded for efficiency. Use awaitGuiInitialized() to await full initialization if necessary.
