Descripción detallada

A set of parameters for statistics in EvA2. Several data entries are provided by the AbstractStatistics class, others by the additional informers. This class allows customization of entries and frequency of data output. Data entries can be selected using a StringSelection instance. There is a switch called "output full data as text" which will be interpreted by AbstractStatistics showing all or only the selected entities.
