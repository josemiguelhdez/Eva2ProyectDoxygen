Descripción detallada

Interface problem class for Matlab(TM). Towards EvA2 this behaves like any other double valued problem implementation. However internally, every evaluation "asks" a mediator instance for the result which waits for Matlab to evaluate the x value. When Matlab is finished, the mediator returns to the evaluate method and the optimization can continue.
