Descripción detallada

Swap random pairs of values of a GA/GI individual. If preferPairs is true, unequal pairs are picked with some preference (by trying for l/2 times, where l is the binary individual length). A range of number of par mutations can be given from which the actual number of pairs is drawn in a uniform way.
