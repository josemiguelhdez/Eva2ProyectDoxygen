Descripción detallada

A synchronous problem changes in equal intervals. As the EA takes notice of the problem only in evaluations, the smallest reasonable interval is once per evaluation so that the problem changes after every evaluation. This of course under the simplification that an evaluation takes no time, or from a different POV, changes within an evaluation can not be taken into account (or just discretely approximated).
