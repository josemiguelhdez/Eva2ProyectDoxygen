Parse a csv file as in the example. Reads columns 2,3,4,5 where counting starts at zero. Checks for the column entries at index 5 to contain a value of 1. These columns are interpreted as target function variables with boundaries and seed. Note that all boundary values must contain valid numbers. If a single seed value is empty, no initialization range is used.

Parámetros
fnamename of the csv data file
Devuelve
an instance of ExternalRuntimeProblem
