Descripción detallada

Interface for dynamic changes to object parameters. When an optimizer has the getter method getParamControl retrieving an instance of this interface, an update is triggered after every optimize call. This allows parameters of the object to be adapted dynamically, such as linearly decreasing weights or control parameters. In case of PSO, this may be the linearly decreasing inertia weight or the activation feedback control mechanism.

Notice how Processor handles parameter control. For now, a controlled object must implement the method getParamControl() which is to return either an InterfaceParameterControl or an array of Objects which should either be instances of InterfaceParameterControl or themselves should implement getParamControl(). This allows recursive parameter adaption hierarchies.
