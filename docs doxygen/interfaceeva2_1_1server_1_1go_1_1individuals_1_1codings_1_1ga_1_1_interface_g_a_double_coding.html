This method decodes a part of a given BitSet into a double value. This method may change the contens of the BitSet if it doesn't describe a valid value. The method checks whether or not the value is within the given range.

Parámetros
refBitSetThe BitSet where the questioned value is stored.
rangeThe allowed range of the value.
locusThe position and length on the BitSet that is to be decoded.
correctionEnable automatic correction is enabled.

The float value.
