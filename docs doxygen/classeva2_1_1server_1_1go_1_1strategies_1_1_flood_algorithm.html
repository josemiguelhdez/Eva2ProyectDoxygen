Descripción detallada

The flood algorithm, and alternative to the threshold algorithms. No really good but commonly known and sometimes even used. Here the problem is to choose the initial flood peak and the drain rate such that it fits the current optimization problem. But again this is a greedy local search strategy. Similar to the evolutionary programming strategy this strategy sets the mutation rate temporarily to 1.0. The algorithm regards only one-dimensional fitness. Created by IntelliJ IDEA.
