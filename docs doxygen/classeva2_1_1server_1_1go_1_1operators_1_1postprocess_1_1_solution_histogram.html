Create a fitness histogram of an evaluated population within the given interval and nBins number of bins. Therefore a bin is of size (upperBound-lowerBound)/nBins, and bin 0 starts at lowerBound. Returns an integer array with the number of individuals in each bin.

Parámetros

lowerBoundlower bound of the fitness interval
upperBoundupper bound of the fitness interval
